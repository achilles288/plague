# Plague

Command line COVID-19 infection tracking tool. Up to date data from world wide.

## Usage

```bash
plague [country]
```

Examples
```bash
plague                # Shows world wide data
plague us             # Shows United States' data
plague myanmar        # Shows Myanmar's data
```


For details,

```bash
plague --help
```
